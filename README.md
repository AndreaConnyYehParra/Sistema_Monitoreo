# Sistema_Monitoreo
Sistema de Monitoreo de la salud de la red del Centro de Cómputo del ITC desarrollado en Python.

La implementación de este proyecto, se explica en los siguiente pasos:

1. Asegurese de estar conectado en la vlan 210
2. Clonar el repositorio del proyecto https://github.com/AndreaConnyYehParra/Sistema_Monitoreo.git
3. Abrir la carpeta del proyecto en Visual Studio Code
4. Importar en XAMPP el archivo de la base de datos nombrada monitoreoitc.sql que se encuentra ubicado en la carpeta database
5. Abrir una terminal en Visual Studio Code
6. Crear el entorno virtual con el comando python -m venv venv
7. Activar el entorno virtual con el comando venv\Scripts\activate
8. Instalar dependencias con el comando pip install -r Sistema_Monitoreo/requirements.txt
9. Ejecuta el proyecto con el comando python .\Sistema_Monitoreo\app.py  
10. Abrir en el navegador http://127.0.0.1:5000
